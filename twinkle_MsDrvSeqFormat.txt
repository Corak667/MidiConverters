MsDRV Sequence Format (for MsDRV 4.4)
=====================
(.MS files)

Header
------
24h*4 bytes - track pointers
3*4 bytes - ?? (always 00)
4 bytes - end of file offset

Sequence Commands
-----------------
Note: All commands are padded up to 4 bytes. The padding byte is 9E.

00-7F dd ll vv - play note with length ll, delay dd, volume vv
81 xx xx xx - ??
82 ii - set FM/SSG Instrument
85 vv - set Volume (00..7F) [unused in MsDRV 4]
	MIDI: Volume defines Note Velocity
	FM: TL operator = (7F - vv)
	SSG: volume = (vv / 08)
8A tt - set Tempo to tt BPM
96 xx xx - ?? (parameters seem to be always 00)
9B tt - Loop End, loop tt times (tt = 0 -> infinite)
9C - Loop Start
9D ii - set Detune
9F pp - set Pan (01 = right, 02 = left, 03 = center)
A4 ll mm - Pitch Bend (mm = semitones, ll = fraction)
A5 xx - ??
C1 xx xx - ??
C2 - set GS checksum marker
C3 dd - send raw MIDI byte dd (used for long SysEx messages)
C4 - send GS checksum (calculated over all bytes sent via command C3 since the last C2 command)
DD dd v1 v2 - SysEx data bytes 1-2, delay dd
DE dd v3 v3 - SysEx data bytes 3-4 and send SysEx command, delay dd
		SC-55 Mode: F0 41 di mi 12 v1 v2 v3 v4 [chksum] F7
DF dd di mi - set SysEx command header (di = device ID, mi = model ID)
E2 dd ii mm - Bank MSB mm, Bank LSB 00, MIDI Instrument ii, delay dd
E6 dd cc - set MIDI Channel to cc, delay dd
E7 dd ?? ?? - ??
EB dd cc vv - MIDI Controller cc, value vv, delay dd
EC dd ii - MIDI Instrument ii, delay dd
EE dd ll mm - MIDI Pitch Bend, delay dd
FE - Track End
